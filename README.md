# 💄 L'Oréal Beauty Assistant - AI Chatbot

> _"Because You're Worth It"_ - An advanced AI-powered beauty consultant representing L'Oréal Paris

A sophisticated, production-ready chatbot that provides personalized beauty advice, product recommendations, and skincare routines using OpenAI's GPT-4o model. Features modern chat interface, conversation context tracking, and premium L'Oréal branding.

## 🚀 **Quick Start**

### **Option 1: Direct Browser**

1. Open `index.html` in any modern web browser
2. Start chatting about beauty topics
3. Experience personalized L'Oréal recommendations!

### **Option 2: Local Server**

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx serve .
```

Then visit: `http://localhost:8000`
